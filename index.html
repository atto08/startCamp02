데이터의 흐름을 지휘하는 논리, 지치지 않는 추진력.

97년생, 역동적인 백엔드 엔지니어 박현도입니다.

필드를 누비는 스포츠의 역동성을 코드 앞에서의 끈기로 치환합니다.
Java와 Spring Boot를 주력 무기로, 복잡한 비즈니스 로직을 단단하게 풀어냅니다.

단순히 돌아가는 코드를 넘어, 왜 이 기술을 선택했는지 명확한 근거를 제시할 수 있는 개발자를 지향합니다.

집요한 원인 분석과
성능 최적화

Karman 프로젝트 V2 최적화 과정에서 단순한 쿼리 튜닝을 넘어섰습니다. 벌크 연산 시 영속성 컨텍스트의 초기화(clearAutomatically) 타이밍과 같은 미세한 원인까지 분석하여 약 78~80%의 실질적인 성능 개선을 이끌어냈습니다.

분산 환경에서의
데이터 정합성 유지

Radiata 프로젝트를 통해 MSA 기반 아키텍처를 설계했습니다. 복잡하게 분산된 서비스 환경 속에서 Saga 패턴 등을 활용하여 데이터의 일관성과 정합성을 유지하는 방법을 치열하게 고민하고 적용했습니다.

흔들리지 않는 멘탈과
문제 해결력

복잡한 2D 배열 시뮬레이션 같은 하드한 알고리즘 문제 앞에서도 포기하지 않습니다. 운동으로 다져진 강인한 멘탈과 긍정적인 에너지는 팀 프로젝트의 예기치 못한 병목 앞에서도 강력한 돌파구가 됩니다.

SSAFY, 우리의 시너지가 시작되는 곳

뛰어난 동료 교육생 여러분과 서로의 개발 노하우를 아낌없이 공유하며 상부상조하고 싶습니다. 우리가 함께 흘리는 땀방울과 코드 리뷰는 가장 확실한 성장의 거름이 될 것입니다.

이곳에서 다 같이 치열하게 성장하여, 목표하는 엔터프라이즈 기업의 문을 함께 두드립시다!
